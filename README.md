# ARC Raider Items Search

Web application for searching ARC Raider items built with React and Vite.

## Features

- Search items by name
- Display item images and rarity
- Responsive design
- Fast and lightweight

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment to GitHub Pages

1. Build the project:
   ```bash
   npm run build
   ```

2. The build files will be in the `dist` folder

3. Deploy the `dist` folder to GitHub Pages

## Data Source

The application loads data from `/public/data.json` which is copied from `../sources/generated.json`
